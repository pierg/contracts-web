Web Interface for the Contract-Based Goal Graph package.

## Installation

With `pip`:
```bash
pip install contracts-web
```

With [`pipx`](https://github.com/pipxproject/pipx):
```bash
python3.7 -m pip install --user pipx
pipx install contracts-web
```
